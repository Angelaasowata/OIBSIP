# Retail Sales EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover trends, customer purchasing behavior, and sales performance using Python.

## Objectives
- Clean and prepare the dataset.
- Analyze sales trends.
- Explore customer demographics.
- Analyze product category performance.
- Visualize insights using charts.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Files
- Retail_Sales_EDA.ipynb
- retail_sales_dataset.csv
- Images

## Conclusion
The analysis provides valuable insights into customer behavior and sales patterns, helping support data-driven business decisions.
