# OIBSIP
Retail Sales EDA: Data Cleaning, Visualization, Trend Analysis &amp; Business Insights using Python.
## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Objectives
- Clean and inspect the dataset
- Analyze sales trends
- Explore customer demographics
- Identify top-performing products
- Generate actionable business insights
