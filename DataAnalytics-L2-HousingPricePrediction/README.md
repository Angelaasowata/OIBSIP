# Housing Price Prediction

## Project Overview
This project predicts house prices using machine learning techniques. A Linear Regression model was developed to analyze housing features and estimate property prices.

## Objectives
- Load and preprocess the housing dataset.
- Explore the data through visualization.
- Train a Linear Regression model.
- Evaluate model performance using MAE, MSE, RMSE, and R² Score.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Files
- Housing_Price_Prediction.ipynb
- Housing.csv
- Images

## Conclusion
The model provides house price predictions based on housing features and demonstrates the application of machine learning in predictive analytics.
